type Matriz a = [[a]] #

Uma posição numa matriz dada como um par (linha,colunha). As coordenadas são dois números naturais e começam com (0,0) no canto superior esquerdo, com as linhas incrementando para baixo e as colunas incrementando para a direita: | Uma matriz é um conjunto de elementos a duas dimensões.

dimensaoMatriz :: Matriz a -> DimensaoMatriz #

Calcula a dimensão de uma matriz.

NB: Note que não existem matrizes de dimensão m * 0 ou 0 * n, e que qualquer matriz vazia deve ter dimensão 0 * 0.
